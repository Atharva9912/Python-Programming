# Python-Programming
### Web Scraping, EDA &amp; Data Visualization of ‘bikewale.com’
- Analyzed a dataset of two-wheeler specifications and customer ratings, actionable insights were extracted about pricing, performance and brand positioning to stakeholders for informed decision making.
- By conducting univariate and bivariate analysis, enabled data-driven insights that support better decision-making for consumers, manufacturers, and retailers in the motorcycle market.
- Languages and Tools used: Python, Web Scraping, Pandas, Numpy, Matplotlib, PowerPoint.
